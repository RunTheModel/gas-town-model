# Modeling Gas Town

These files are attempt to capture the complexity of gas town in a new formal modeling language (Kinner).

The Kinner compiler has yet to be released but these artifacts were generated by pointing it at the gas-town repo and distilling out these files as a first stab at a model.  More to come, but the most interesting thing right now is the graphvize diagram depicting a convoy landing a pair of issues (convoy-lands.dot).  That file is a graphviz diagram - open it in a DOT visualizer to see a depiction of a a mayor landing a Convoy, with hooks, polecats, etc all depicted working together.
